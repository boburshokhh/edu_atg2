# Документация: Универсальный компонент станций

## Описание

Создан универсальный компонент `StationDetail.vue`, который работает для всех 8 компрессорных станций СП ООО «ASIA TRANS GAS». Компонент автоматически отображает детальную информацию о любой станции на основе её ID.

## Структура данных

### Файл данных: `src/data/stationsData.js`

Этот файл содержит централизованную базу данных всех станций. Каждая станция имеет следующую структуру:

```javascript
{
  id: 1,                        // Уникальный ID станции
  name: 'Полное название',      // Полное наименование
  shortName: 'WKC1',            // Краткое название
  description: 'Описание...',   // Детальное описание
  image: 'WKC1.jpg',           // Фото станции (в /public/stations/)
  techMapImage: '/tex_kart/...', // Технологическая карта
  power: '25 МВт',             // Мощность
  commissionDate: '25.11.2009', // Дата ввода в эксплуатацию
  coursesCount: 8,              // Количество тренингов
  status: 'active',             // Статус (active/maintenance)
  location: 'Адрес...',         // Местоположение
  type: 'Тип станции...',       // Тип КС
  designCapacity: '32 млрд м³/год', // Проектная мощность
  gasPressure: '7.5 МПа',       // Давление газа
  
  // Основное оборудование
  equipment: [
    {
      name: 'Название',          // Наименование оборудования
      model: 'Модель',           // Тип/Модель
      manufacturer: 'Производитель', // Производитель
      quantity: 3                // Количество
    }
  ],
  
  // Технические характеристики

    {
      category: 'Категория',     // Название параметра
      value: '9-12',             // Значение
      unit: 'млн м³/сут',        // Единица измерения
      description: 'Описание'    // Описание (опционально)
    }
  ],
  
  // Системы безопасности
  safetySystems: [
    {
      name: 'Название системы',
      description: 'Описание функционала',
      manufacturer: 'Производитель' // опционально
    }
  ],
  
  // Нормативные документы
  normativeDocs: [
    'ГОСТ 5542-2014 ...',
    'СП 36.13330.2012 ...'

```

## Доступные станции

1. **WKC1** - Головная компрессорная станция (ID: 1)
2. **WKC2** - Промежуточная КС (ID: 2)
3. **WKC3** - Промежуточная КС (ID: 3)
4. **UCS1** - Дожимная КС (ID: 4)
5. **UCS3** - Дожимная КС (ID: 5)
6. **GCS** - Газокомпрессорная станция (ID: 6)
7. **MS** - Модульная станция (ID: 7)
8. **UKMS** - Узбекская КС (ID: 8)

## Компонент StationDetail.vue

### Основные секции:

1. **Hero Section**
   - Фоновое изображение станции
   - Основная информация (название, описание)
   - Краткие характеристики (мощность, дата, статус, тренинги)

2. **Техническая информация (ГОСТ стиль)**
   - Общие сведения
   - Основное оборудование (таблица)
   - Технические характеристики
   - Системы безопасности и контроля
   - Нормативные документы

3. **Технологическая карта**
   - Отображение технической схемы
   - Возможность увеличения (lightbox)
   - Предупреждение о конфиденциальности

4. **Блок тренингов**
   - Информация о доступных программах
   - Кнопка перехода к тренингам

### Особенности:

- ✅ Универсальный шаблон для всех станций
- ✅ Автоматическое отображение данных по ID
- ✅ Адаптивный дизайн (mobile, tablet, desktop)
- ✅ Профессиональное оформление по ГОСТу
- ✅ Условное отображение секций (если данные есть)

## Маршрутизация

```javascript
// router/index.js
{
  path: '/station/:id',
  name: 'StationDetail',
  component: StationDetail,
  props: true
}
```

### Примеры URL:
- `/station/1` - WKC1
- `/station/2` - WKC2
- `/station/8` - UKMS

## Как добавить новую станцию

1. Откройте `src/data/stationsData.js`
2. Добавьте новый объект с уникальным ID:

```javascript
export const stationsData = {
  // ... существующие станции
  
  9: {
    id: 9,
    name: 'Новая компрессорная станция',
    shortName: 'NKS',
    description: 'Описание новой станции...',
    image: 'NKS.jpg',
    // ... остальные поля
  }
}
```

3. Добавьте фото в `/public/stations/NKS.jpg`
4. Добавьте технологическую карту в `/public/tex_kart/` (если есть)
5. Обновите список станций в `Stations.vue`

## Как обновить данные существующей станции

1. Откройте `src/data/stationsData.js`
2. Найдите нужную станцию по ID
3. Обновите необходимые поля
4. Сохраните файл

Компонент автоматически отобразит обновленные данные!

## Технологии

- **Vue 3** - Composition API
- **Vue Router** - Динамическая маршрутизация
- **Tailwind CSS** - Стили
- **vue-easy-lightbox** - Просмотр изображений

## Дизайн

- Корпоративные цвета ATG (синий, белый)
- Профессиональное оформление по ГОСТу
- Адаптивная верстка
- Анимации и hover-эффекты
- Тени и градиенты для глубины

## Примечания

- Все данные хранятся в одном файле для удобства управления
- Компонент универсален и работает для всех станций
- Условное отображение секций (v-if) для гибкости
- Технологические карты могут быть общими или уникальными
- Фотографии используются одинаковые для демонстрации (можно добавить уникальные)
