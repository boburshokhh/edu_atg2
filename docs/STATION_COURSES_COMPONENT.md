# Компонент StationCourses - Программа обучения станции

## Описание

Универсальный и гибкий компонент для отображения образовательной программы онлайн-тренингов по компрессорным станциям. Дизайн вдохновлен лучшими практиками Udemy, Coursera и Skillshare, но реализован в уникальном минималистичном стиле с возможностью масштабирования.

## Особенности

✅ **Современный дизайн** в стиле ведущих образовательных платформ
✅ **Полностью адаптивный** для всех устройств
✅ **Аккордеон-структура** для удобного изучения программы
✅ **Гибкая архитектура данных** для легкого добавления новых курсов
✅ **Интерактивные элементы** с плавными анимациями
✅ **SEO-оптимизированная структура** с breadcrumbs

## Структура компонента

```
src/
├── views/
│   └── StationCourses.vue          # Главный компонент курсов станции
├── data/
│   └── stationsData.js             # Данные курсов для каждой станции
└── router/
    └── index.js                    # Роут /station/:id/courses
```

## Навигация

### URL-структура
```
/stations                           → Список всех станций
/station/:id                        → Детальная информация о станции
/station/:id/courses                → Программа обучения станции
```

### Пример
```
/station/1                          → Станция WKC1
/station/1/courses                  → Программа обучения WKC1
```

## Использование

### 1. Добавление курсовой программы для станции

В файле `src/data/stationsData.js` добавьте объект `courseProgram` к данным станции:

```javascript
{
  id: 1,
  name: 'Компрессорная станция WKC1',
  // ... другие поля
  
  courseProgram: {
    title: 'Программа онлайн-тренинга "Компрессорная станция WKC-1"',
    description: 'Описание курса...',
    duration: '10 академических часов',
    topicsCount: 34,
    lessonsCount: 4,
    testsCount: 5,
    format: 'Онлайн',
    
    learningOutcomes: [
      'Результат обучения 1',
      'Результат обучения 2',
      // ...
    ],
    
    requirements: [
      'Требование 1',
      'Требование 2',
      // ...
    ],
    
    targetAudience: [
      'Целевая аудитория 1',
      'Целевая аудитория 2',
      // ...
    ],
    
    lessons: [
      {
        title: 'Урок № 1: Название урока',
        duration: '2.5 часа',
        topics: [
          { 
            code: 'Тема 1.1', 
            title: 'Название темы', 
            duration: '35 мин' 
          },
          // ...
        ],
        materials: ['Форум', 'Дополнительные материалы'],
        test: {
          title: 'Тестовые задания к Уроку 1',
          questions: 5
        }
      },
      // ...
    ]
  }
}
```

### 2. Навигация к курсам

Из любого места приложения используйте:

```javascript
// Переход к программе курсов станции
this.$router.push(`/station/${stationId}/courses`)

// или с использованием именованного роута
this.$router.push({ 
  name: 'StationCourses', 
  params: { id: stationId } 
})
```

### 3. Кнопки навигации

В `Stations.vue` и `StationDetail.vue` уже настроены кнопки:

```vue
<button @click="$router.push(`/station/${station.id}/courses`)">
  Тренинги
</button>
```

## Структура данных

### CourseProgram Object

| Поле | Тип | Описание |
|------|-----|----------|
| `title` | String | Название программы тренинга |
| `description` | String | Краткое описание программы |
| `duration` | String | Общая продолжительность (напр. "10 академических часов") |
| `topicsCount` | Number | Количество тем в программе |
| `lessonsCount` | Number | Количество уроков |
| `testsCount` | Number | Количество тестов |
| `format` | String | Формат обучения (напр. "Онлайн") |
| `learningOutcomes` | Array<String> | Массив ожидаемых результатов обучения |
| `requirements` | Array<String> | Массив требований к участникам |
| `targetAudience` | Array<String> | Массив целевой аудитории |
| `lessons` | Array<Lesson> | Массив уроков программы |

### Lesson Object

| Поле | Тип | Описание |
|------|-----|----------|
| `title` | String | Название урока |
| `duration` | String | Продолжительность урока |
| `topics` | Array<Topic> | Массив тем урока |
| `materials` | Array<String> | Дополнительные материалы |
| `test` | Test | Объект тестового задания |

### Topic Object

| Поле | Тип | Описание |
|------|-----|----------|
| `code` | String | Код темы (напр. "Тема 1.1") |
| `title` | String | Название темы |
| `duration` | String | Продолжительность темы |

### Test Object

| Поле | Тип | Описание |
|------|-----|----------|
| `title` | String | Название теста |
| `questions` | Number | Количество вопросов |

## UI Компоненты

### Hero Section
- Градиентный фон с информацией о курсе
- Breadcrumb навигация
- Кнопка "Назад к станциям"
- Основные характеристики курса (длительность, темы, уроки, тесты)
- CTA кнопки "Начать обучение" и "Скачать программу"

### Main Content
**Левая колонка (2/3 ширины):**
- Блок "Что вы изучите" (learningOutcomes)
- Программа курса с аккордеонами для уроков
- Требования к участникам
- Целевая аудитория

**Правая колонка (1/3 ширины, sticky):**
- Карточка записи на курс
- Блок "Курс включает"
- Кнопки социальных сетей

### Аккордеон уроков
- Красивый дизайн с номерацией
- Плавное раскрытие/скрытие тем
- Отображение дополнительных материалов
- Кнопка прохождения теста
- Hover-эффекты для интерактивности

## Стилизация

### Цветовая палитра
- **Градиенты:** `from-blue-600 via-blue-700 to-indigo-800`
- **Акценты:** Синий `blue-600`, Зеленый `green-500`, Янтарный `amber-500`
- **Фон:** Светло-серый `gray-50`
- **Карточки:** Белый с тенями

### Анимации
```css
/* Раскрытие аккордеона */
.expand-enter-active,
.expand-leave-active {
  transition: all 0.3s ease;
  max-height: 2000px;
  overflow: hidden;
}

.expand-enter-from,
.expand-leave-to {
  max-height: 0;
  opacity: 0;
}
```

4. Убедитесь, что `stationId` передается корректно
